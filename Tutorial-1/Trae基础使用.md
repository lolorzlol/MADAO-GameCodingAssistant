## 计划模式、构建模式、Solo模式

### 模式概览

```
┌─────────────────────────────────────────────────────────┐
│                    Trae 双重开发模式                      │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   IDE 模式                                    Solo 模式 │
│   ┌─────────────────┐                         ┌─────────┐│
│   │  Chat 模式      │                         │         ││
│   │  智能问答辅助    │                         │ 自主开发 ││
│   │  代码解释/修复   │                         │ 全流程  ││
│   ├─────────────────┤                         │ 代理    ││
│   │  Builder 模式   │                         │         ││
│   │  项目自动生成    │                         │         ││
│   │  需求→代码      │                         │         ││
│   └─────────────────┘                         └─────────┘│
│                                                         │
│   开发者控制流程               AI自主完成全流程           │
└─────────────────────────────────────────────────────────┘
```

### Chat 模式（智能问答）

**适用场景**：日常编码辅助、代码解释、Bug 修复

**核心能力**：
- 上下文锁定：AI 理解整个项目上下文
- 多模态交互：截图上传分析架构图、异常堆栈
- 代码生成：自然语言描述生成代码

**使用方式**：
- 快捷键：`Ctrl+U`（Windows）/ `Command+U`（macOS）
- 点击右上角聊天图标

### Builder 模式（项目构建）

**适用场景**：从零创建项目、快速原型开发

**核心能力**：
- 自然语言描述需求 → AI 自动生成完整项目结构
- 自动安装依赖、配置环境
- 对话迭代优化

**操作流程**：
```
1. 点击右侧「Builder」标签
      ↓
2. 输入需求描述（如"创建 Python 贪吃蛇游戏"）
      ↓
3. AI 生成代码文件、依赖清单、资源文件
      ↓
4. 点击「运行」自动安装依赖
      ↓
5. 对话迭代优化
```

### Solo 模式（自主开发代理）

**适用场景**：全流程自主开发、复杂项目实现

**核心特点**：
- AI 作为自主开发代理，独立完成从需求分析到部署的全流程
- 适合开发者完全放手让 AI 主导的场景

| IDE 模式 | Solo 模式 |
|----------|-----------|
| 开发者控制关键流程 | AI 自主完成全流程 |
| 实时建议和补全 | 自然语言需求 → 完整项目 |
| 适合日常开发迭代 | 适合快速原型/新项目 |

## 子代理创建

### 智能体架构

Trae 采用**多智能体协同架构**，将传统 IDE 从"工具集合"升级为"AI 团队协作平台"。

```
用户请求
    ↓
┌─────────────────────────────────────┐
│           主代理 (Main Agent)        │
│         任务分析和分配协调            │
└─────────────────────────────────────┘
    ↓
    ├── 子代理 1 (代码生成)
    ├── 子代理 2 (代码审查)
    ├── 子代理 3 (测试执行)
    └── 子代理 4 (Bug 修复)
    ↓
整合结果 → 输出给用户
```
