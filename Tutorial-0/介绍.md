# 为什么选择Trae
原则：优先有免费模型，其次才是好用。
对比Trae, Qwen Code, Opencode, Claude Code
## Trae
优点：免费模型多且模型强度高，上手快
缺点：不支持插件
## Qwen Code
优点：有一个免费模型且不限速
缺点：不支持插件，mcp安装有限制，网络搜索有限制
## Opencode
优点：支持插件，mcp
缺点：免费模型不稳定
## Claude
优点：最好用
缺点：完全没有免费模型

# 免费额度体验
1. 阿里百炼新用户免费
2. 硅基流动有免费额度，另外有永久免费的9B以下小模型
3. 智谱新用户额度多(GLM-5)
4. Minimax新用户也有免费额度

# 为什么选择Unity为主
1. 小团队做独立游戏UE本来就很少
2. UE用AI需要会代码，蓝图AI无法生成。
3. 其他内容都一样，主要的区别其实只有mcp部分。
